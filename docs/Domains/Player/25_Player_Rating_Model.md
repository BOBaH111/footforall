# 25. Player Rating Model

## Purpose

Player Rating Model определяет архитектурную модель цифрового рейтинга игрока.

Рейтинг является объективной оценкой игровых навыков игрока, основанной исключительно на подтверждённых игровых данных.

Рейтинг никогда не изменяется вручную.

Все изменения производятся исключительно Rating Engine.

---

# Principles

Player Rating должен быть:

- объективным;
- воспроизводимым;
- объяснимым;
- независимым от субъективных оценок;
- пересчитываемым при изменении алгоритмов.

---

# Source of Truth

Единственным источником изменения рейтинга являются:

- Evidence Package;
- Match Trust Score;
- Snapshot Player Profile.

Никакие внешние данные не могут изменить рейтинг напрямую.

---

# Rating Components

Player Rating состоит из нескольких независимых компонентов.

```
Overall Rating

=

Position Skill

+

Experience Modifier

+

Confidence

+

Activity Modifier

+

Time Modifier
```

Каждый компонент рассчитывается независимо.

---

# Position Skill

Основной показатель игровых навыков.

Для каждой позиции хранится собственный рейтинг.

Например:

- Goalkeeper
- Defender
- Midfielder
- Forward

Игрок может развивать любое количество позиций одновременно.

---

# Overall Rating

Overall Rating представляет собой агрегированную оценку уровня игрока.

Он рассчитывается автоматически на основе Position Ratings.

Overall Rating не изменяется напрямую.

---

# Experience Modifier

Experience отражает объём подтверждённых игровых данных.

Например:

- количество сыгранных минут;
- количество завершённых матчей;
- количество турниров;
- количество различных соперников.

Experience не показывает силу игрока.

Он показывает зрелость оценки.

---

# Confidence

Confidence определяет степень доверия системы к рассчитанному рейтингу.

Например:

```
Overall Rating

92

Confidence

18%
```

означает:

Система предполагает высокий уровень игрока, однако объём подтверждённых данных пока недостаточен.

Другой пример:

```
Overall Rating

92

Confidence

98%
```

означает:

Рейтинг подтверждён большим количеством игровых данных.

Confidence никогда не уменьшает Rating напрямую.

Он является самостоятельной характеристикой качества оценки.

---

# Activity Modifier

Игрок, регулярно участвующий в матчах, предоставляет системе актуальные игровые данные.

Длительное отсутствие новых матчей снижает актуальность оценки.

Activity Modifier отражает текущую активность игрока.

---

# Time Modifier

Даже высокий рейтинг постепенно теряет актуальность без новых подтверждений.

Time Modifier рассчитывается Time Evolution Engine.

Таким образом рейтинг постепенно адаптируется к текущему уровню игрока.

---

# Match Trust

Каждый расчёт рейтинга учитывает Match Trust Score.

Игровые действия, полученные в матчах с более высокой степенью доверия, оказывают большее влияние на рейтинг.

Данный механизм определяется ADR-008 — Match Trust Principle.

---

# Position Ratings

Каждая игровая позиция развивается независимо.

Например:

```
Goalkeeper

61

Defender

79

Midfielder

86

Forward

91
```

Игрок может изменить игровую специализацию без создания нового профиля.

---

# Best Position

Система автоматически определяет лучшую позицию игрока.

Лучшая позиция определяется на основе:

- Position Rating;
- количества игровых минут;
- объёма подтверждённых данных.

Best Position вычисляется автоматически.

Пользователь не может выбрать её вручную.

---

# Rating Evolution

Каждое изменение рейтинга представляет собой Evolution Result.

Например:

```
Forward Rating

84

↓

85

Reason

Goal

Trust

0.96
```

История изменений сохраняется полностью.

---

# Rating Decay

При отсутствии новых подтверждённых матчей Time Evolution Engine постепенно снижает актуальность рейтинга.

Decay никогда не происходит мгновенно.

Изменение выполняется плавно.

---

# Rebuild

Player Rating может быть полностью пересчитан.

```
Player Created

↓

Evidence Replay

↓

Rating Engine

↓

Current Rating
```

Это обеспечивает возможность изменения алгоритмов без потери цифровой истории.

---

# Long-term Vision

Player Rating является динамической математической моделью, а не фиксированным числом.

Рейтинг отражает не только предполагаемый уровень игровых навыков, но и степень уверенности системы в этой оценке.

Таким образом цифровая футбольная репутация игрока строится на объективных игровых данных, подтверждённых завершёнными матчами, а не на субъективных оценках пользователей.
