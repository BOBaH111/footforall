# Architecture Decision Records

Architecture Decision Records (ADR) содержат все архитектурные решения, принятые в ходе разработки Footforall.

Каждый ADR фиксирует:

- контекст принятия решения;
- принятое решение;
- последствия;
- возможные направления дальнейшего развития.

ADR не являются техническим долгом.

Они являются архитектурной памятью проекта и позволяют понимать причины существующих решений спустя годы после их принятия.
