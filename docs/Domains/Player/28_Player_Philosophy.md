# 28. Player Philosophy

## Purpose

Player является центральной сущностью цифровой футбольной экосистемы Footforall.

Матчи, турниры, площадки, судьи и остальные компоненты платформы существуют для формирования объективной цифровой футбольной истории игрока.

Главной ценностью платформы является не матч, а эволюция игрока.

---

# Player First

Все основные процессы платформы в конечном итоге направлены на изменение Player Profile.

Матч существует как источник объективных игровых данных.

Player Profile является конечной целью этих данных.

---

# Digital Identity

Player Profile представляет собой цифровую футбольную личность пользователя.

Она развивается на протяжении всей футбольной карьеры.

Ни один матч не является самоцелью.

Каждый матч становится частью цифровой истории игрока.

---

# Objective Evolution

Игрок развивается исключительно на основании объективных игровых данных.

Субъективные оценки не являются источником эволюции Player Profile.

---

# Continuous Growth

Каждый завершённый матч является новым этапом развития.

Даже если рейтинг игрока не изменился, матч может изменить:

- опыт;
- активность;
- репутацию;
- достижения;
- доверие системы;
- статистику.

Таким образом каждый матч создаёт новую ценность.

---

# Football Memory

Платформа хранит не только текущее состояние игрока.

Она хранит всю историю его футбольной жизни.

История никогда не удаляется.

Каждый матч остаётся частью цифровой памяти игрока.

---

# Multiple Roles

Player является одной из возможных ролей пользователя.

User может одновременно обладать несколькими независимыми ролями.

Каждая роль имеет собственный профиль.

---

# Explainability

Любое изменение Player Profile должно быть объяснимо.

Игрок должен иметь возможность понять:

- почему изменился рейтинг;
- почему изменилась репутация;
- какие игровые события повлияли на изменение.

Ни одно изменение не должно быть «магическим».

---

# Reproducibility

Любое состояние Player Profile может быть воспроизведено.

Для этого достаточно:

- истории матчей;
- утверждённых протоколов;
- Evidence;
- используемой версии алгоритмов.

---

# Fairness

Все игроки оцениваются одинаковыми алгоритмами.

Алгоритмы не зависят от личности игрока.

Изменения Player Profile всегда основаны на одинаковых правилах.

---

# Long-term Vision

Player Profile является цифровым футбольным паспортом пользователя.

С каждым новым матчем этот паспорт становится точнее.

Ценность платформы определяется не количеством пользователей и не количеством матчей.

Главная ценность Footforall — объективная цифровая футбольная история каждого игрока.
