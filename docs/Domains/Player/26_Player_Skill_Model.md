# 26. Player Skill Model

## Purpose

Player Skill Model определяет цифровую модель игровых навыков игрока.

Skill Model описывает не общий рейтинг игрока, а его отдельные игровые качества.

Каждый навык развивается независимо и рассчитывается автоматически на основе объективных игровых данных.

---

# Principle

Игровой навык является производной величиной.

Навык никогда не изменяется вручную.

Все изменения рассчитываются исключительно Skill Engine.

---

# Source of Truth

Источниками изменения Skill являются:

- Evidence Package;
- Match Trust Score;
- Snapshot Player Profile.

---

# Skill Model

Player Skill состоит из независимых категорий.

Например:

```
PACE

SHOOTING

PASSING

DRIBBLING

DEFENDING

PHYSICAL
```

Список категорий может расширяться.

---

# Skill Categories

## Pace

Отражает способность игрока быстро перемещаться по полю.

Примеры Evidence:

- количество скоростных рывков;
- успешные выходы один на один;
- участие в быстрых атаках.

---

## Shooting

Отражает качество завершения атак.

Примеры Evidence:

- голы;
- точность ударов;
- реализация голевых моментов.

---

## Passing

Отражает способность создавать моменты для партнёров.

Примеры Evidence:

- голевые передачи;
- точность передач;
- передачи под удар.

---

## Dribbling

Отражает способность сохранять владение мячом.

Примеры Evidence:

- успешные обводки;
- продвижение мяча;
- выход из-под прессинга.

---

## Defending

Отражает эффективность оборонительных действий.

Примеры Evidence:

- успешные отборы;
- перехваты;
- блокированные удары.

---

## Physical

Отражает способность сохранять эффективность в течение матча.

Примеры Evidence:

- сыгранные минуты;
- интенсивность участия;
- единоборства;
- устойчивость к нагрузке.

---

# Position Independence

Каждый Skill рассчитывается независимо от игровой позиции.

Например:

Нападающий также может обладать высоким Defending.

Защитник может иметь высокий Passing.

---

# Position Rating

Position Rating рассчитывается на основе комбинации отдельных Skill.

Таким образом Position Rating является производным показателем.

Skill остаются первичным уровнем модели.

---

# Overall Rating

Overall Rating рассчитывается на основе Position Ratings.

```
Skill

↓

Position Rating

↓

Overall Rating
```

---

# Evolution

Каждый Skill развивается независимо.

После каждого матча Skill Engine анализирует соответствующие Evidence и формирует Evolution Result.

---

# Confidence

Каждый Skill имеет собственный уровень Confidence.

Например:

```
Passing

86

Confidence

95%
```

```
Dribbling

91

Confidence

42%
```

Таким образом система знает, насколько достоверно оценён каждый навык.

---

# Time Evolution

При отсутствии новых подтверждённых данных Time Evolution Engine постепенно снижает актуальность Skill.

Все изменения выполняются плавно.

---

# Long-term Vision

Skill Model позволяет объективно описывать сильные и слабые стороны игрока.

Player Rating становится производной величиной, основанной на независимых игровых навыках.

Это делает цифровую модель игрока более объяснимой, прозрачной и пригодной для дальнейшего развития.
