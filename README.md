# ⚽ Footforall

> Digital Football Ecosystem

## Mission

Create the world's leading digital football ecosystem where every completed match becomes part of a player's football history.

## Core Idea

Footforall is not a match-finding service.

It is a platform that builds a player's digital football reputation.

Every completed match creates new value for every participant.

## North Star Metric

Completed Matches
