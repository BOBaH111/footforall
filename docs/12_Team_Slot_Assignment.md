# Team Slot Assignment Domain

## Назначение

Team Slot Assignment связывает игрока с конкретным игровым слотом в рамках матча.

Сущность хранит историю того, какой игрок занимал конкретную позицию в определённый момент времени.

Все игровые события, статистика и рейтинг игроков рассчитываются на основании истории назначений слотов.

---

# BR-SLOT-001. Назначение игрока

Игрок может быть назначен только на свободный игровой слот.

После назначения создаётся TeamSlotAssignment.

---

# BR-SLOT-002. Один активный слот

В каждый момент времени игрок может занимать только один игровой слот.

---

# BR-SLOT-003. Один игрок на слот

В каждый момент времени игровой слот может быть занят только одним игроком.

---

# BR-SLOT-004. Замена игрока

Замена не изменяет игровой слот.

При замене:

- текущее назначение завершается;
- создаётся новое назначение для того же слота.

---

# BR-SLOT-005. История назначений

История назначений не изменяется.

После завершения назначения создаётся новое назначение.

Редактирование завершённых назначений запрещено.

---

# BR-SLOT-006. Игровые события

Игровые события всегда принадлежат игроку.

Игровой слот используется только для определения позиции игрока на момент события.

---

# BR-SLOT-007. Изменение позиции

Если игрок во время матча меняет игровую позицию, текущее назначение завершается.

Создаётся новое назначение с новым игровым слотом.

---

# BR-SLOT-008. Минуты на поле

Продолжительность нахождения игрока на поле рассчитывается как сумма всех его назначений.

---

# BR-SLOT-009. Завершение матча

После завершения матча все активные назначения автоматически завершаются.

После этого история назначений становится неизменяемой.
